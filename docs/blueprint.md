# **App Name**: Resonant Harmony

## Core Features:

- User Authentication: Secure user signup and login with Firebase Authentication.
- Product Catalog: Display a catalog of singing bowls with images, descriptions, and prices.
- Add to Cart: Functionality to add selected singing bowls to a shopping cart.
- View/Manage Cart: Allows users to view and manage items in their shopping cart.
- Checkout Process: Enable users to complete the order
- Bowl Recommendation: Suggest the ideal Singing Bowl based on the visitor's intentions, using a tool informed by current inventory.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke calmness and nature.
- Background color: Desaturated light brown (#F5F5DC) for a warm, grounding feel.
- Accent color: Soft gold (#DAA520) for highlights and CTAs.
- Body and headline font: 'Alegreya', a serif, for an elegant, intellectual feel.
- Use simple, nature-inspired icons for navigation and actions.
- Clean, spacious layout with a focus on product imagery.
- Subtle transitions and animations for a smooth user experience.
